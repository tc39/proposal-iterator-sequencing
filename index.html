    Iterator Sequencing

    1 Iterator.concat ( ...items )

    1. Let closure be a new Abstract Closure with no parameters that captures items and performs the following steps when called:
      1. Repeat, while items is not empty,
        1. Let iter be the first element of items.
        2. Remove the first element from items.
        3. Let iteratorRecord be ? GetIteratorFlattenable(iter, reject-strings).
        4. Let innerAlive be true.
        5. Repeat, while innerAlive is true,
          1. Let innerValue be ? IteratorStepValue(iteratorRecord).
          2. If innerValue is done, then
            1. Set innerAlive to false.
          3. Else,
            1. Let completion be Completion(Yield(innerValue)).
            2. If completion is an abrupt completion, then
              1. Set completion to Completion(IteratorClose(iteratorRecord, completion)).
              2. Repeat, while items is not empty,
                1. Let trailingIter be the first element of items.
                2. Remove the first element from items.
                3. If trailingIter is an Object, then
                  1. Let nextMethod be Completion(Get(trailingIter, "next")).
                  2. If nextMethod is an abrupt completion, then
                    1. If completion is not a throw completion, set completion to nextMethod.
                  3. Else,
                    1. Set nextMethod to ! nextMethod.
                    2. If IsCallable(nextMethod), then
                      1. Let record be the Iterator Record { [[Iterator]]: iter, [[NextMethod]]: nextMethod, [[Done]]: true }.
                      2. Set completion to Completion(IteratorClose(iteratorRecord, completion)).
              3. Return ? completion.
      2. Return ReturnCompletion(undefined).
    2. Return CreateIteratorFromClosure(closure, "Iterator Helper", %IteratorHelperPrototype%, « »).
